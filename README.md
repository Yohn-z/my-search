## 极简个人搜索页
### 在线访问
https://explore.yohn-z.cn/my-search/
### 技术栈
使用原生三剑客（html+css+js）开发 , 无需编译部署
### 目录结构说明
* /css  - css样式
* /icon  - 图标
* /img  - 图片
* /js  - js脚本
* /favicon.svg  - 网站图标、页签图标
* /index.html  主页
